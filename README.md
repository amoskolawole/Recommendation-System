# Recommendation-System

A user-item behavior(collaborative filtering) ML movie recommendation system using 'userId','movieId' and 'rating' feature to find similar movies between different users with the help of sklearn metrics 'cosine similarity'.

## Objectives
- Data inspection
- Data Combination/Merging
- Data Transformation to Pivot Table
- NaN Columns Auto filling
- Creating Recommendation Function
- Manual Testing

## Tools & Library Used
- Python
- Pandas
- Scikit-learn
- Cosine Similarity
- Joblib

## ML Workflow
- Dataset Merging
- Cosine Similarity
- Joblib Utilization

## Findings
In collaborative filtering recommendation, dataset need to be transform to pivot table for cosine similarity to be able to detect similarities across users preference movie.
